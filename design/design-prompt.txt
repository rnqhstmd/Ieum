[서비스 한 줄]
Google 로그인 기반의 노션(Notion) 유사 "실시간 협업 문서 서비스".
개인/공유 워크스페이스 · 중첩 페이지 · 블록 에디터 · 실시간 공동편집(라이브 커서·접속자 표시).

[핵심 제약 — 반드시 지킬 것]
- 웹 서비스 전용입니다. 네이티브 앱 없음.
- 반응형 필수: 데스크탑 + 모바일 웹 둘 다 완벽 동작.
  · 데스크탑(>=1024px): 좌측 고정 사이드바 + 우측 메인의 2-pane.
  · 태블릿(768~1023px): 사이드바 좁게 또는 토글.
  · 모바일(<768px): 사이드바는 햄버거→드로어로 숨김, 에디터 풀폭,
    모달은 풀스크린/바텀시트, 터치 타겟 충분히, presence는 압축 표시.
- 프론트 스택은 Next.js(App Router) + React + Tailwind CSS.
  → 이 스택으로 구현 가능한 범위로, 컴포넌트 단위 재사용 가능하게 디자인.
- 비주얼 무드/색/타이포 톤은 함께 첨부되는 design.md를 따른다.
  (이 프롬프트는 "구조·기능·동작" 명세이고, 무드는 design.md가 담당.)

--------------------------------------------------------------------
[화면 목록 — 각 화면마다 데스크탑/모바일 모두 제시]
--------------------------------------------------------------------
1. 랜딩(비로그인): 제품 소개 히어로 + "Google로 시작" CTA. 간결.
2. 로그인: Google 로그인 버튼 1개(백엔드 OAuth2로 이동). 미니멀.
3. 앱 셸(로그인 후 공통 레이아웃): 좌측 사이드바 + 우측 메인 콘텐츠.
   모바일에서는 상단바 햄버거로 사이드바 드로어 토글.
4. 사이드바:
   - 워크스페이스 스위처(개인 1개 + 공유 워크스페이스 목록, 전환)
   - 페이지 트리(중첩, 펼침/접힘 토글, 들여쓰기로 계층 표현)
   - "새 페이지" / "새 워크스페이스" 버튼
   - 하단 계정 영역(아바타 → 메뉴: 로그아웃)
5. 페이지(에디터) — 가장 중요한 화면:
   - 상단: 페이지 아이콘(이모지) + 제목(인라인 편집)
   - 본문: 블록 에디터(단락 / 제목1~3 / 글머리 기호 목록)
   - 자동저장 상태 표시(예: "저장됨")
   - 우상단: 접속자 아바타 스택 + "N명 보는 중"
   - 본문 내: 협업자 라이브 커서(이름 라벨 + 고유 색상)
6. 공유 워크스페이스 멤버 관리(모달 또는 설정 패널):
   - 멤버 목록(역할 뱃지: OWNER=관리자 / MEMBER)
   - 이메일 초대 입력 + 보내기
   - 보류 중 초대 목록(취소 버튼)
   - 멤버 역할 변경 / 내보내기 (※ OWNER에게만 보임)
7. 초대 수락 화면: "OOO 워크스페이스에 초대되었습니다" + 수락 버튼.
   만료/이미 멤버/유효하지 않은 초대 상태 처리.
8. 페이지 컨텍스트 메뉴: 이름변경 / 아이콘 변경 / 하위 페이지 추가 / 아카이브(삭제).
9. 상태 화면: 로딩 스켈레톤, 빈 워크스페이스/빈 페이지, 오프라인·재연결 배너,
   에러 토스트, 권한 없음(403).

--------------------------------------------------------------------
[기능 동작 — 어떻게 동작해야 하는지]
--------------------------------------------------------------------
- 인증: 비로그인으로 보호 화면 접근 시 로그인으로 유도. 로그인 성공 →
  개인 워크스페이스 루트(대시보드)로 이동.
- 개인 워크스페이스: 가입 시 자동 1개 생성. 추가/삭제 불가, 이름변경만.
- 페이지: 무한 중첩. 부모를 펼치면 자식 표시. 삭제는 아카이브(soft delete).
  제목은 인라인 편집, 아이콘은 이모지.
- 에디터: Enter로 새 블록, 빈 블록에서 Backspace로 삭제/이전 블록과 병합,
  블록 타입 전환 가능. 입력은 즉시 반응(낙관적) + 자동저장.
- 실시간 협업: 같은 페이지를 여러 명이 동시에 편집해도 충돌/덮어쓰기 없이
  양쪽이 같은 내용으로 수렴. 다른 사람이 위쪽을 편집해도 내 커서/상대 커서가
  올바른 위치를 유지. 접속/이탈 시 아바타와 "보는 중" 표시가 실시간 갱신.
- 권한: 공유 워크스페이스는 OWNER(관리자)만 초대·멤버 제거·역할 변경·
  워크스페이스 삭제·이름변경 가능. MEMBER는 페이지 편집만. 권한 없는
  사용자에게는 관리 UI를 숨기거나 비활성화.
- 초대: OWNER가 이메일로 초대 → 메일 발송 → 수락 시 멤버 합류.
  초대는 7일 후 만료, 취소 가능, 이미 멤버면 거절 처리.
