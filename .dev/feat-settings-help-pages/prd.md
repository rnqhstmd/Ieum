# PRD: AccountMenu 설정·도움말 페이지 구현 및 배선

## 배경
사이드바 계정 메뉴(AccountMenu)에 설정/테마/도움말/로그아웃 4개 항목이 있으나 **설정·도움말은 클릭해도 반응이 없다**(AccountArea 핸들러가 빈 함수). 테마·로그아웃은 정상 동작. 계정 메뉴는 이미 getCurrentUser(name/email), logout(→/login), 테마 토글(dataset.theme + localStorage 'ieum-theme')을 보유. `(app)` 라우트 그룹은 AppShell 셸 안에 렌더. 디자인 시안 없음(콘텐츠 신규 정의).
**주의**: 현재 base(main)에선 사이드바 "⌘K"가 비기능 placeholder(커맨드 팔레트 배선 #43은 미머지). 도움말에 미동작 단축키 안내 시 주의(BR-2).

## 목표
계정 메뉴 설정·도움말 클릭이 실제 페이지 이동으로 이어지게 하고, 설정에서 계정확인·테마·로그아웃, 도움말에서 사용 안내를 제공한다.

## 요구사항
### [Must]
- FR-1: 계정 메뉴 "설정" 클릭 → `/settings` 이동.
- FR-2: 계정 메뉴 "도움말" 클릭 → `/help` 이동.
- FR-3: 두 페이지 모두 기존 사이드바 셸 안에서 렌더.
- FR-4: 설정 페이지가 로그인 계정의 이름/이메일 표시(조회만).
- FR-5: 설정 페이지에서 테마(다크/라이트) 전환.
- FR-6: 설정 페이지에서 로그아웃.
- FR-7: 도움말 페이지가 정적 안내 콘텐츠 표시. (범위 = 스코프 Q2)
### [Should]
- FR-8: 미로그인 상태로 /settings·/help 직접 접근 시 /login 이동.
### 비즈니스 규칙
- BR-1: 설정 계정 정보는 읽기 전용(수정 기능 없음).
- BR-2: 도움말은 실제 동작 기능만 안내. 미동작 기능은 미동작 표시 또는 제외. (처리 = 스코프 Q2)

## 수용 기준 (G-W-T)
- AC-1: Given 계정 메뉴 열림 / When "설정" 클릭 / Then `/settings`로 router.push 호출 [FR-1]
- AC-2: Given 계정 메뉴 열림 / When "도움말" 클릭 / Then `/help`로 router.push 호출 [FR-2]
- AC-3: Given 계정 조회 성공 / When 설정 페이지 렌더 / Then 이름·이메일 텍스트가 화면에 나타남 [FR-4]
- AC-4: Given 설정 페이지 렌더 / When 계정 정보 영역 확인 / Then 이름/이메일 입력창·저장 버튼이 존재하지 않음 [BR-1]
- AC-5: Given 설정 페이지 렌더 / When 테마 전환 컨트롤 클릭 / Then 테마 상태·저장값 갱신 + 테마 표시 텍스트(다크/라이트) 변경 [FR-5]
- AC-6: Given 설정 페이지 렌더 / When 로그아웃 버튼 클릭 / Then logout() 호출 후 `/login` router.push [FR-6]
- AC-7: Given 도움말 페이지 접근 / When 렌더 / Then 앱 소개 안내 콘텐츠 텍스트(제목/설명 등)가 화면에 나타남 [FR-7]
- AC-8: Given 도움말 페이지 렌더 / When 콘텐츠 확인 / Then 미동작 단축키(⌘K 등) 안내가 포함되지 않는다(단축키 목록 미노출) [BR-2]
- AC-9: Given 계정 조회 401 실패 / When /settings·/help 접근 / Then `/login`으로 router.push [FR-8]

## 제외 범위
- 계정명/비밀번호 변경 등 설정 값 수정(BR-1).
- 워크스페이스 관리/멤버 초대(이미 /workspace/{id}/members 존재).
- ⌘K 커맨드 팔레트 실제 동작화(이번 범위 아님).

## 스코프 확정 (사용자 선택)
- Q1 설정 콘텐츠: **계정 정보(이름/이메일 읽기전용) + 테마 전환 + 로그아웃**.
- Q2 도움말 콘텐츠: **앱 소개(정적)만. 단축키(⌘K 등) 미노출**(#43 미머지로 비동작 → 안내 안 함).
- Q3 네비게이션: **전체 페이지 이동**(사이드바 유지, 메인 영역 전환).
- Q4 로그인 필요: **둘 다 로그인 필요**((app) 그룹, 미인증 401 시 /login).
